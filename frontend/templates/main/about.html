About LogiComp

Our Mission

The LogiComp is designed to provide a comprehensive solution for organizing and participating in programming competitions. Our goal is to make it easy for educators and organizations to host coding contests while providing participants with a seamless experience.

Features

  • Create and manage programming contests with multiple problems
  • Automatic judging of submissions with detailed feedback
  • Real-time leaderboards to track progress
  • Secure authentication and participant management
  • Support for multiple programming languages

Technology Stack

Backend
  • Python with Flask framework
  • PostgreSQL database
  • SQLAlchemy ORM
Frontend
  • HTML5, CSS3, JavaScript
  • Bootstrap 5 for responsive design
  • Jinja2 templating
